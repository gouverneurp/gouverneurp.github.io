eVAS: A user-friendly electronic Visual Analogue Scale

This is the download page of the electronic visual Analogue Scale (eVAS). It is an open source, easy to use and user friendly slider that records the slider values and saves them to CSV files. The config.ini file can be edited to customise the eVAS to your needs. Possible errors are logged in a log.txt file. More information can be found under github.com/gouverneurp/eVAS.
